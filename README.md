## 파일 작성 및 commit 규칙
- 파일 명: 플랫폼_문제이름_닉네임.py
- 파일 예시:

```
PGS_교점에별만들기_디노.py"
```

- 플랫폼 작성법 통일: 
  * [BOJ] - 백준 
  * [SWEA] - 삼성SW Expert Academy
  * [PGS] - 프로그래머스
  * [ETC] - 그외


- commit 메세지: "feat: n주차 - 닉네임"
- description: 문제 주소 (option)
- 터미널에서 작성법: 

```
git commit -m "feat: 1주차 - 디노"
```
